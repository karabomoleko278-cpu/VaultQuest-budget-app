# VaultQuest - Premium Budget Tracker

VaultQuest is a sophisticated Android application designed to help users track their expenses, manage categories, and set financial goals with a premium user experience.

## Features

- **Secure Authentication**: User-specific data storage with login/registration.
- **Category Management**: Custom categories for organized tracking.
- **Detailed Entries**: Record expenses with date, start/end times, and descriptions.
- **Photo Attachments**: Optionally attach photos of receipts or items to each entry.
- **Goal Tracking**: Set minimum and maximum monthly spending goals using interactive SeekBars.
- **Comprehensive Reports**: 
  - View full history for user-selectable periods.
  - View category-wise totals for selected periods.
- **Local Persistence**: Powered by RoomDB for reliable offline data storage.

## Technical Details

- **Language**: Kotlin
- **Persistence**: Room Database (SQLite)
- **UI Architecture**: Multiple Activities with Intent-based navigation.
- **Styling**: Material Design 3 with a custom premium theme.
- **Testing**: JUnit for logic verification.
